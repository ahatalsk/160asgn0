Notes to grader: For the "awesomeness" part of this assignment, I added a few extra features to the paint program. I added two extra shapes, a star and a heart, that are both composed of triangles. I also added three new buttons, an "Undo" button, a "Set Background" button, and a "Random" button. The undo button removes the last shape drawn on the canvas and can be clicked repeatedly to remove multiple shapes. The set background button sets the canvas color to the color that is currently selected on the RGB sliders and clears the canvas. When the "Clear" button is pressed, the background reverts back to black. The random button randomizes the drawing tool. It will pick a random color, size, and shape and change the sliders accordingly to reflect the new drawing mode.

Please use a browser that supports "canvas"
